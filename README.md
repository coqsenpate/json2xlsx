# json2xlsx

## Description
Converts a json file to xslx. 

## Usage
  * Install Nodejs.
  * Install coffeescript:  
  `npm install -g coffee-script`
  * Install npm dependencies:  
  `npm install`
  * Run the script:
  `coffee app.coffee input_file_path [output_file_path]`
